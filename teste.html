seletores mais usados

seletores

os seletores sao implementados via css, para estilizar o html, como "atalho" para cada seletor, como h1, header, class, id, p, etc

seletor universal

Representado por um *, para aplicar estilos a todos os elementos do documento

seletor de tipo

Representa um elemento HTML específico, como h1, p, div, etc. Ele seleciona todos os elementos do tipo especificado.

seletor de classe

Representado por um ponto (.), seguido pelo nome da classe. Ele seleciona todos os elementos que possuem a classe especificada.

seletor de ID

Representado por uma cerquilha (#), seguido pelo nome do ID. Ele seleciona um elemento único com o ID especificado.
